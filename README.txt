RUNAR SIGNATURE – CLOUD HOSTING (GitHub Pages)

1) Lag et nytt GitHub-repo, f.eks. `runar-signature-site`.
2) Last opp mappen `assets/` og filen `signature_cloud_template.html` til repoet.
3) Aktiver GitHub Pages i repoet (Settings → Pages → Deploy from a branch, velg `main`).
4) Nettadresse blir: https://<BRUKERNAVN>.github.io/runar-signature-site
5) Åpne `signature_cloud_template.html` og erstatt {{BASE_URL}} med denne nettadressen.
6) Kopier HELE innholdet og lim inn i nye Outlook → ⚙️ → E-post → Skriv og svar → Signatur.
